# כנרת 2026 — גרסת לייב עם Firebase

זה פרויקט מוכן לאפליקציה חיה:
- כמה אנשים יכולים להיכנס לאותו חדר
- כל שינוי מתעדכן בזמן אמת
- יש חדרים עם קוד
- ציוד / חברים / רעיונות / הוצאות

## מה יש בתוך התיקייה
- `index.html` — המסך הראשי
- `styles.css` — עיצוב
- `app.js` — הלוגיקה של האפליקציה
- `firebase.json` — קובץ הגדרה לאחסון האתר
- `firestore.rules` — חוקי הגישה למסד הנתונים
- `.firebaserc.example` — דוגמה לקישור לפרויקט

## מה אתה צריך לעשות
1. לפתוח Firebase Console
2. ליצור פרויקט חדש
3. להדליק:
   - Authentication → Anonymous
   - Firestore Database
   - Hosting
4. להדביק את נתוני `firebaseConfig` בתוך `index.html`
5. להעלות את הקבצים ל-Firebase Hosting

## שים לב
כרגע הגישה לחדרים היא לפי קוד חדר בלבד.
זה טוב להתחלה מהירה, אבל לא אבטחה קשוחה.
