# הוראות מלאות בעברית — שלב אחרי שלב

## המטרה
להעלות את האפליקציה לאינטרנט בחינם בצורה הכי פשוטה, כדי שכל חבר יוכל להיכנס עם קוד חדר ולראות שינויים בזמן אמת.

---

## שלב 1 — פתח פרויקט Firebase
1. היכנס ל-Firebase Console
2. לחץ `Create a project`
3. תן שם לפרויקט, לדוגמה:
   `kinneret-2026-live`
4. אפשר לבטל Google Analytics אם לא צריך
5. צור את הפרויקט

---

## שלב 2 — הפעל Firestore
1. בתפריט הצד בחר `Firestore Database`
2. לחץ `Create database`
3. בחר `Start in production mode`
4. בחר מיקום קרוב, לדוגמה:
   `europe-west`
5. אשר

אחרי זה:
1. עבור ללשונית `Rules`
2. הדבק את התוכן של הקובץ `firestore.rules`
3. לחץ `Publish`

---

## שלב 3 — הפעל Authentication
1. בתפריט הצד בחר `Authentication`
2. לחץ `Get started`
3. עבור ללשונית `Sign-in method`
4. הפעל `Anonymous`
5. שמור

---

## שלב 4 — קח את firebaseConfig
1. במסך הראשי של הפרויקט לחץ על גלגל שיניים → `Project settings`
2. גלול למטה לקטע `Your apps`
3. לחץ על סמל `</>` כדי להוסיף Web app
4. תן שם לאפליקציה, לדוגמה:
   `kinneret-live-web`
5. אשר
6. Firebase יציג לך אובייקט שנראה כך:

```js
const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
```

7. פתח את הקובץ `index.html`
8. חפש את השורה:
   `window.FIREBASE_CONFIG = { ... }`
9. הדבק שם את הנתונים האמיתיים שלך במקום `PASTE_HERE`

---

## שלב 5 — התקנת Firebase CLI
במחשב עם Node.js מותקן:

```bash
npm install -g firebase-tools
```

בדיקה:
```bash
firebase --version
```

אם אין לך Node.js:
1. תתקין Node.js מהאתר הרשמי
2. ואז תריץ שוב את הפקודה למעלה

---

## שלב 6 — כניסה ל-Firebase
```bash
firebase login
```

ייפתח דפדפן. תאשר את החשבון שלך.

---

## שלב 7 — חבר את התיקייה לפרויקט שלך
1. פתח את הקובץ `.firebaserc.example`
2. שנה את:
   `YOUR_FIREBASE_PROJECT_ID`
   לשם הפרויקט האמיתי שלך
3. שמור בשם:
   `.firebaserc`

דוגמה:
```json
{
  "projects": {
    "default": "kinneret-2026-live"
  }
}
```

---

## שלב 8 — פרסום האתר
עמוד בתוך התיקייה של הפרויקט ואז תריץ:

```bash
firebase deploy
```

אחרי ההעלאה Firebase יחזיר כתובת אתר, בדרך כלל בצורה:
```text
https://your-project-id.web.app
```

זאת הכתובת שאתה שולח לחברים.

---

## שלב 9 — שימוש בפועל
1. אתה נכנס לאתר
2. כותב את השם שלך
3. לוחץ `צור חדר חדש`
4. מתקבל קוד חדר
5. אתה שולח לחברים:
   - את כתובת האתר
   - את קוד החדר
6. כל אחד נכנס, כותב שם, מצטרף לחדר
7. כל שינוי מתעדכן אצל כולם בזמן אמת

---

## מה עובד כרגע
- יצירת חדר
- הצטרפות לחדר עם קוד
- שמירת חברים
- רשימת ציוד
- שחרור/לקיחה של ציוד
- רעיונות
- הוצאות
- עדכון שם חדר ותיאור
- סנכרון חי

---

## מה עדיין בסיסי
- אין הרשאות מתקדמות לפי משתמש
- אין כניסה עם גוגל/טלפון
- אין עריכת פריט קיים
- אין מחיקת חדר שלם
- אין ניהול תפקידים מנהל/רגיל

---

## מה כדאי לעשות אחר כך
אם תרצה שדרוג:
1. כניסה עם Google
2. הרשאות מנהל
3. עריכה מלאה של פריטים/הוצאות/חברים
4. קישור הזמנה אוטומטי עם קוד כבר בפנים
5. PWA — שיהיה כמו אפליקציה בטלפון
6. לוגו ומיתוג מקצועי

---

## אם יש שגיאה
### אם האתר נטען אבל לא מתחבר
בדוק:
- שהפעלת `Anonymous Auth`
- שהדבקת נכון את `firebaseConfig`
- שפרסמת את `firestore.rules`

### אם אתה רואה הודעת Permission denied
זה כמעט תמיד אומר:
- שהחוקים לא פורסמו
- או ש-Auth לא הופעל

### אם לא רואים שינויים בזמן אמת
בדוק:
- שאתה וכל החברים באותו קוד חדר
- שהאינטרנט מחובר
- שאין שגיאה ב-console

---

## חשוב
הפרויקט הזה בנוי להתחלה חינמית ופשוטה.
אם תהיה תנועה גדולה מאוד או שימוש כבד, ייתכן שבשלב מסוים תצטרך לעבור למסלול בתשלום.
